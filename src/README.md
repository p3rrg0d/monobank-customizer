# Модульна структура коду

Код editor.js тепер розділений на модулі для кращої організації та підтримки:

## Структура файлів

```
src/
├── utils.js           - Допоміжні функції (hexToRgba)
├── GradientPicker.js  - Клас для редагування градієнтів
├── WidgetEditor.js    - Основний клас редактора
└── main.js            - Точка входу (entry point)
```

## Опис модулів

### `utils.js`
- **hexToRgba()** - Конвертує HEX колір в RGBA з альфа-каналом

### `GradientPicker.js`
- Управління градієнтними зупинками (stops)
- Рендеринг UI для редагування градієнта
- Drag & drop для точок градієнта
- Інтеграція з Pickr для вибору кольорів

### `WidgetEditor.js`
- Головний контролер редактора
- Управління станом віджета
- Ініціалізація всіх компонентів
- Генерація CSS коду
- Draggable логіка для віджета

### `main.js`
- Точка входу
- Ініціалізація WidgetEditor при завантаженні DOM

## Використання

HTML завантажує main.js як ES module:
```html
<script type="module" src="src/main.js"></script>
```

Всі модулі використовують ES6 import/export синтаксис.
