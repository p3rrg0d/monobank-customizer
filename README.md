# Редактор віджета Monobank

Простий та зручний редактор для кастомізації віджета Monobank.  
Виконаний у стилі **retro + neobrutalism** та дозволяє швидко налаштовувати дизайн, кольори, шрифти та інші параметри віджета без зайвих складнощів.

---

## Особливості

1. **Інтерактивний редактор** – миттєво переглядайте зміни CSS-коду.  
2. **Гнучке налаштування кольорів** – створюйте власну кольорову схему віджета.  
3. **Експорт CSS в один клік** – швидко отримуйте готовий код для вставки на сайт.
